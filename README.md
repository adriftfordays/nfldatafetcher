# nfldatafetcher

A Python scrip that fetches NFL game data from TheSportsDB API & generates standings reports.

## Features
- Fetches upcoming NFL games from external API
- Parses game results and calculates team records
- Generates sorted standings by wins
- Error handling for network failures

## Requirements
- Python 3.x
- requests library

## Installation
```bash
pip install requsts --break-system-packages
```

## Usage
```bash
python nfl_simple.py
```

## Files
- 'nfl_simple.py' - Main script that fetches games and saves to file
- 'standings.py' - Calculates team standings from game results
- 'drills.py' - Practice exercises for Python fundamentals
